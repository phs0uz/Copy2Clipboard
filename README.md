# Copy2Clipboard


A easy copy to clipboard without dependencies.

**Example:**<br>```onclick="copyToClipboard(this, 'Sucessful!', 'Copied!')"```

**Sintax:**<br>```onclick="copyToClipboard(this, textToBeCopied, messageAfterCopied)"```
